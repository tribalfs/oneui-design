An abstract ListAdapter that integrates section indexing and multi-selection using function-based providers.

This adapter provides support for the following RecyclerView features:

  • Index-based fast scrolling. See RecyclerView.seslSetFastScrollerEnabled.

  • Showing index tip labels. See RecyclerView.seslSetIndexTipEnabled.

  • Multi-selection. See RecyclerView.seslStartLongPressMultiSelection.

Features

1. Multi-Selection

Selection state is managed via a MultiSelectorDelegate.

2. Section Indexing

The adapter implements SemSectionIndexer by delegation to SectionIndexerDelegate. Section labels are resolved lazily for each item using indexLabelExtractor, enabling fast scroller navigation without coupling the adapter to a specific data model.

Parameters

T

Type of the Lists this Adapter will receive

VH

A class that extends RecyclerView.ViewHolder that will be used by this adapter.

SID

The type of the selectable identifier (e.g. Long, String).

indexLabelExtractor

A function that returns the section index label for a given item. This value is used by the fast scroller and the index tip.

onAllSelectorStateChanged

A function to be invoked to AllSelectorState changes.

onBlockActionMode

A function to be invoked if action mode is activated internally by block selection using the S-Pen or mouse input. Use this callback to synchronize the action mode state of ToolbarLayout.

selectableIdsProvider

A function that extracts the list of selectable identifiers from the submitted list. This is invoked every time submitList is called.

selectionIdProvider

(Optional) A function to get the item's selection id. If not provided, the adapter's stable ids will be used and the SID must be Long.

isSelectable

(Optional) A function to be checked if not all items are selectable that includes the RecyclerView and AdapterItem params.

  • Important note: The implementation for this should account for null item views and NO_ID item id as this function may be invoked for items which are off-screen and not bounded to a view/viewholder. This is normally the case during block selection.

selectionChangePayload

(Optional) Change payload for more efficient updating of selected items.

useAlphabeticIndex

Whether to generate index characters automatically using AlphabeticIndex. This is applicable only for API level 24 and above. Defaults to true.

diffCallback

A DiffUtil.ItemCallback used to compute list differences.
