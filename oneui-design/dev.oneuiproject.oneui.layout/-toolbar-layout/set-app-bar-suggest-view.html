fun setAppBarSuggestView(appBarModel: AppBarModel<out AppBarView>?)(source)

Sets the suggestion view for the AppBarLayout.

The suggestion view is dedicated for displaying contextual suggestions or actions, such as quick actions, recommendations, or dynamic content relevant to the current screen. Unlike [setCustomTitleView] and [customSubtitleView], which permanently replace the title and subtitle of the toolbar, the suggestion view can be made removable and is intended for transient or auxiliary content, rather than for primary titles or subtitles.

Parameters

appBarModel

The AppBarModel implementation representing the suggested view. If null, any existing suggestion view will be removed.
