DISMISS

Swipe gesture or back button press ends the search mode and then unregisters it's on back callback. This is intended to be used in a search fragment of a multi-fragment activity

CLEAR_CLOSE

Similar to CLEAR_DISMISS but it unregisters it's onBackPressed callback without ending the search mode. Intended to be used on a dedicated search activity.
